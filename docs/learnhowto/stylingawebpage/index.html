Styling A Web Page

A web page without style isn't much to look at. Lets add some styling to the page we created in the creating a web page section.

CSS Or Sass

Trio supports both plain CSS and Sass, including both the old and new Sass dialects. Whether you prefer CSS or Sass, old or new dialects, is of course entirely up to you but for the purpose of this exercise we are going to use Sass to add some styling to our page.

  1. In the source/sass folder create a new file named main.scss.

  2. Using your editor of choice, open the file an copy/paste the following into it.

     h1 {
     color: darkgreen;
 }

  3. Build the project. From the project's root folder run the following from the command line.

     trio build

    Once the build is complete, open up your project's public/css folder. You should see 2 new files there - main-map.css.map and main.18db23283b81cf71a699eb58cef946cb.css.

  4. Now we need to link public/css/main.18db23283b81cf71a699eb58cef946cb.css up into our web page. Open the file source/templates/default.html in your editor and add the following to the bottom of the head section

     <link rel="stylesheet" href="/css/main.18db23283b81cf71a699eb58cef946cb.css">

    which should then look like this:

     <head>
     <meta charset="UTF-8">
     <meta name="viewport" content="width=device-width, initial-scale=1.0">
     <meta http-equiv="X-UA-Compatible" content="ie=edge">
     <title>title</title>
     <link rel="stylesheet" href="/css/main.18db23283b81cf71a699eb58cef946cb.css">
 </head>

  5. Now lets build and run our website. From the project's root folder run the following from the command line

     trio build && trio serve

    and after Trio has built and launched the website in the browser you should see a dark green "Hello World" in all its glory. Yeaaaaaa!

Note: For those who prefer to work with plain CSS all you would need to change is to create a new file in source/css named main.18db23283b81cf71a699eb58cef946cb.css and of course add the styling to the h1 element there. Everything else is the same and when you build the project and open public/css you would see a new file there named main.18db23283b81cf71a699eb58cef946cb.css.
