Get Started

Follow the 6 steps below to quickly get up and running.

  • Step 1.

    Install Node

    If you don't already have Node installed, please go to nodejs.org and follow the installation instructions you find there.

  • Step 2.

    Install Trio

    To install Trio, run the following from the command line:

    npm i -g @4awpawz/trio
  • Step 3.

    Create A New Project

    To create a new Trio project, run the following from the command line:

    trio new path/to/new/project

    where "path/to/new/project" is the path to where you want to create the new project.

  • Step 4.

    Build A Project

    To build a project, from the project's root folder run the following from the command line:

    trio build
  • Step 5.

    Run A Project

    To run a project, from the project's root folder run the following from the command line:

    trio serve
